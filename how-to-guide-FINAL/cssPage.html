CSS

Before we wrap up and call it the end lets add some CSS to the web pages. To add CSS to the web pages, create a folder called “static” inside the “mysite” folder, inside it, create a folder called “css” and inside it - create a file called indexPage.css. Inside it, lets add some styles for the buttons and the title: 
               
                body{
                    text-align: center;
                    background-color: beige;
                }
                
                body h1{
                    font-size: 72px;
                }
                
                body a button {
                    width: 400px;
                    height: 200px;
                    border: none;
                    background-color: aqua;
                    color: white;
                    font-size: 48px;
                }
Here are some quick styles I threw on-to the page, feel free to experiment at this point and add styles to all pages on the website!
